一个短视频单体项目，使用七牛云OSS、CDN等服务。主要包括用户、视频、内容审核、视频推送、评论等模块。

技术栈为：SpringBoot、Mybatis、MySQL、Redis、JWT、Nginx、七牛云、阿里云、ApiFox、Vue、Git、Arthas、Coze
